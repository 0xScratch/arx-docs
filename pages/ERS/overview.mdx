# Overview

The Ethereum Reality Service ("ERS") is an open protocol that enables resolution of chips, dapp links and smart contrcts. It attests to the nature of a given chip manufacturer as well as the brand or creator adding the chip to a given object. ERS is currently in development for release Q2 2023. You can read more about ERS and submit feedback on [Ethereum Magicians](https://ethereum-magicians.org/t/a-proposal-for-the-ethereum-reality-service/9694).
